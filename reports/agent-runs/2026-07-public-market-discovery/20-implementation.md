## Decisions

- Public capture uses only `fapi.binance.com` and `api.binance.com` hosts per hard constraints.
- Committed fixtures include hand-crafted edge-case symbols (`TSLAUSDT`, `PERPONLYUSDT`, `SPOTONLYUSDT`, `STALEUSDT`) alongside representative `BTCUSDT`/`ETHUSDT` rows.
- UI loads `backend/tests/fixtures/candidate-classification.json` (generated by `run_discovery(use_fixtures=True)`) so the prototype works offline without a live capture.
- Planning preview in the UI mirrors domain logic client-side for simulation display; authoritative logic remains in `backend/domain/planning.py` and is covered by pytest.
