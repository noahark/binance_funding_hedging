## Key verified conclusions

- Margin: `GET /sapi/v1/margin/allPairs` and `/sapi/v1/margin/isolated/allPairs`
  require an API key (HTTP 400, code `-2014` without key). Phase 1 does not use
  them. `margin_public.source = "unverified"` and
  `margin_public.public_cross_margin_pair = null` for all rows. `MARGIN_SPOT_CANDIDATE`
  uses only the public spot field `isMarginTradingAllowed`.
- Funding: `nextFundingTime` is unambiguous (ms epoch of next settlement).
  `lastFundingRate` is documented as "most recently updated funding rate"; the
  settled-vs-estimate distinction is `ambiguous` and must not be labeled as a
  guaranteed settled or upcoming value.
- bStock: `contractType == "TRADIFI_PERPETUAL"` -> `asset_tag = "BSTOCK"`. All
  118 observed TRADIFI symbols are tokenized equities/ETFs/commodities and have
  no spot leg, so they are `PERP_ONLY_EXCLUDED` + `BSTOCK`, proving `asset_tag`
  is independent of `route_class`.
